## Why Cybersecurity

I've spent years writing software, but I kept running into the same gap in my knowledge: I could build systems, but I didn't fully understand how to break them. That matters. Writing secure code requires knowing what attackers actually do, not just following a checklist of best practices.

## The Program

Northwestern's cybersecurity program covers network security, penetration testing, SIEM tools, and defensive strategy over six months of intensive coursework. The curriculum is hands-on — labs, capture-the-flag exercises, real tool usage — which is what sold me. I've done enough self-study to know that reading about Wireshark is not the same as using it to trace a suspicious packet across a network.

## What I'm After

My goal is specific: I want to combine software development with offensive security. Most developers treat security as an afterthought. Most security professionals don't write production code. I want to sit at that intersection — building tools and understanding the threats they face.

I'll be documenting what I learn here, post by post, as I work through the program. The next several months should be interesting.
